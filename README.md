# 视频倍速时长 Chrome 扩展

这个 Chrome 扩展会显示一个小标签：

* 当前倍速
* 按当前倍速折算后的实际观看时长

例如：

* `10:00` 在 `1.25x` 下会显示为 `8:00`

## 文件说明

* `manifest.json`：Manifest V3 配置
* `content.js`：监听视频事件，并按平台选择显示位置
* `content.css`：历史遗留样式文件，当前主 UI 已不再依赖它

## 在 Chrome 中加载

1. 打开 `chrome://extensions/`
2. 开启右上角 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择当前项目里的 `extension/` 目录
5. 如果已经加载过旧版本，先在扩展页点击刷新，再重新刷新 Bilibili 或 YouTube 页面

## 计算方式

扩展会根据当前视频的倍速和总时长计算：

* `实际观看时长 = 原时长 / 当前倍速`

## 说明

* 这个版本面向标准 HTML5 `video` 播放器，因此覆盖范围比站点定制方案更广。
* 在 YouTube、Bilibili、腾讯视频、爱奇艺、优酷这类主流平台，会优先把标签挂到播放器控制栏区域。
* 其他站点会退回到视频左上角的悬浮提示，鼠标移入视频时显示，移出后隐藏。
* 原生浏览器 `controls` 不能被直接修改，所以主流平台也是通过站点播放器 DOM 做贴边注入，不保证所有页面结构都完全一致。
* 某些 DRM、Canvas 播放器或高度定制的视频播放器，可能无法完整支持。
